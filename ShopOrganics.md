## General Info
The aim of creating this e-commerce application is to provide the users a fruitful experience of purchasing all the organic products at one place. As most of the people are becoming more conscious of their health, this application would be the one stop solution to all the products they need. Furthermore, saving the users time in searching for each organic products over various other applications.
## Technologies
The front end and the back end technologies used are as follows:
Front End
This is an single page application and uses react, react-router-dom, create-react-app, babel, webpack and uuid libraries. This application also uses HTML and CSS to specify the content and formatting of the web page.
Back End
The backend/service code uses express-based NodeJS, involves calling REST-based service calls and uses the following libraries: cookie-parser, uuid, nodemon (development only) and babel

## Functionalities
User should first login where username as dog, empty space and special characters are not allowed. Furthermore, password with empty space is also not allowed.
User will be taken to the home page where the products are displayed and clicking on the products takes them to the detail page where quantity can be choosen and added to the cart. User can view previous ratings and reviews given by other users, can also add ratings and also review the product(if they are want to).

Users can also search and sort for products in the home page.

#### Navigation:
Clicking on the 'title'(Shop Organics) takes the user to home page and clicking on 'Cart' displays the products they have in cart(user can delete the items if needed by making the quantity to 0). Users can checkout by clicking on the checkout button in 'Cart'. Users can also view their previous orders if any under 'My orders'.

Furthermore, users can logout by clicking on 'Logout' and if the same user logs in again then they can view their previous orders, if any. Furthermore, they can also view the items in the cart, if any.

## Launch

The project runs with npm install and npm start (doesn't need create-react-app to run with npm start).
